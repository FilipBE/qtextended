Build system changes for 4.2.2

Overview

This page describes changes to the 4.2.2 build system.

  • make sdk fails consistently in every directory if -sdk was not passed.
  • License handling actually works now.
  • New hint=content for adding content items to the PREFIX database.
  • Better warnings/errors for projects that aren't built in singleexec mode.

See also What's New?.

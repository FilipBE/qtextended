Build system changes for 4.4.0

Overview

This page describes changes to the 4.4.0 build system.

  • If you depends() on a project and it is not listed in PROJECTS, your project will be disabled. For compatibility with trees that do not use PROJECTS, the project will be assumed working if a .pro file is found and the PROJECTS variable is not set after reading projects.pri.
  • Plural words are translated for the string language.
  • general.pri and the related no_general_pri CONFIG value have been removed.

See also What's New?.

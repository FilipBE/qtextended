CONFIG Values

Overview

The CONFIG variable has special uses. The follwing values are significant when assigned to it.

CONFIG

quicklaunchNotify the build system that your application supports being built as a quicklaunch plugin.
singleexecNotify the build system that your application should be built into the singleexec binary

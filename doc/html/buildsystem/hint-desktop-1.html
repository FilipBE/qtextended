hint=desktop

Usage

    foo.files=example.desktop
    foo.path=/apps/Applications
    foo.hint=desktop

Description

This is used to specify .desktop files that should have their contents put into the PREFIX database.

Note that you can set .categories here but they will be ignored by content_installer. Edit the .desktop file if you want to specify additional categories.

See also Install Hints, hint=nct, hint=content, and A note on packages.

Image, prefix and SDK location

Qt Extended has separate locations for the image, the prefix and the SDK location.

The image is created when qbuild image is run. The prefix is the location where Qt Extended will be run from. The SDK location is populated as Qt Extended is built. Running qbuild sdk will turn the SDK into a standalone SDK that can be deployed onto other machines.

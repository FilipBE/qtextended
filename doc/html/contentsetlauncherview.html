Detailed Description

The ContentSetLauncherView class displays a content set received via the "ContentSetView" service.

Any entity that creates this widget should show it on reception of the showContentSetView() message on the QPE/System qcop channel. Currently this view is only deployed if Qt Extended has been configured with a drm agent.

This class is part of the Qt Extended server and cannot be used by other Qt Extended applications.

See also LauncherView and ContentSetViewService.

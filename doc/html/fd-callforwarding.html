Communications: Telephony: Call Management: Call Forwarding
Functional Description

The Call Forwarding application allows the user to set how the forwarding of calls is handled under various circumstances.

Features

  • Call Forwarding

Call Forwarding

Call Forwarding allows incoming calls to be forwarded depending upon
any of the following conditions:

  • Always forward all calls
  • Only forward calls when phone is busy
  • Only forward calls when call is unanswered
  • Only forward calls when signal is unavailable

For each of the above conditions it is possible to specify forwarding of calls according to the type of call:

  • Voice calls
  • Data calls
  • FAX calls
  • SMS calls

The user is:

  • provided with a previously used number if it exists.
  • able to forward a call to any number stored in Contacts.
  • able to forward a call to an arbitrary number that is typed in.
  • provided with a message when forwarding fails.

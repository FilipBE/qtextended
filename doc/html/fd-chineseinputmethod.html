Keypad Pinyin Input

Note: This is a sample Chinese input method, and is not intended to be shipped on production devices.

The keypad pinyin input method allows the user to input Chinese characters using a phone keypad. Input is performed on a character-by-character basis using pinyin.

The input sequence is as follows:
1. The user enters a sequence on the keypad. As they are typing, all possible pinyin candidates for that sequence are displayed.
2. The user confirms a pinyin candidate, and character candidates for that pinyin are displayed.
3. The user selects a character, which is sent to the target application.

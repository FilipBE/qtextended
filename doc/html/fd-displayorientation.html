User Interface: Display: Display Orientation
Functional Description

Qt Extended supports 0, 90, 180, and 270 degree screen orientations.

Features

  • Device display rotation support

Device display rotation support

All applications rotate upon request. This can improve the utility of display space on devices such as the N810 or the FIC Freerunner.

The user can set the rotation via the rotation Settings application and applications can change the rotation via a convenient API function.

The rotation API supports the following features:

  • get current rotation
  • change the rotation
  • notification about changes via ValueSpace
  • default rotation (primary rotation based on device key positions)

By using the API, the display can be rotated automatically in response to hardware events such as sliding keyboards or tilt sensors.

The application launcher grid is rotated on a per item basis and not the entire launcher grid widget. This avoids bad appearances when the grid has a bigger height than width in combination with a screen that has a bigger width than height.

Helix DNA Client Integration

Helix DNA Client multimedia codec library is integrated with Qt Extended, allowing access to the many codecs available for Helix.

Helix DNA Client Version 3,1 (Atlas)

Qt Extended uses Helix DNA Client 3.1 (Atlas).

TCK Compliance

Qt Extended passes the RealNetworks' Helix Client Code TCK (excluding device-specific tests).

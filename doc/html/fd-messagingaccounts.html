Messaging: Messaging Accounts
Functional Description

It is possible for a user to view email sent to multiple mail servers. For example the user is able to view messages sent both to a work email address and to a personal email address.

Features

  • Account Management

Account Management

The following account features are available:

  • Multiple Accounts: for retrieving messages from multiple servers.
  • Email Protocols: POP and IMAP for message retrieval and SMTP for message sending.
  • Telephony Protocols: SMS and MMS.
  • Delete: it is possible to delay server deletion of mails until they have been emptied from the messaging trash folder on the device.
  • SMTP: it is possible to use additional SMTP accounts to allow for more than one profile.
  • Signature: it is possible to define a signature block, automatically appended to each message sent with an account.

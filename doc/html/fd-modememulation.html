Communications: Telephony: Modem Emulation
Functional Description

A Qt Extended phone can be used as a modem (for example, from a laptop via Bluetooth LAP/DNP profiles).

Features

  • AT Command Parser
  • Data Calls

AT Command Parser

The AT command parser listens on a serial port, or serial abstraction
(via Bluetooth, Infrared, etc).

Commands are implemented based on the facilities in the telephony library,
so it works in a standards-compliant manner regardless of the back-end modem type.

Data Calls

Data pass through is provided for data and fax calls.

GPRS emulation is provided, which uses the handset's configured network connectivity,
plus IP masquerading, to extend the network to the laptop.

Multimedia: Music Player
Functional Description

A media player focused on the playback of audio and display of audio file information (metadata)

Features

  • Floating player controls
  • View Album contents with Flow View
  • View Albums with Artist Album View
  • View Artists with Artist List View
  • View current acitve playlist with Now Playing View

Floating player controls

This feature is provided as part of the Deskphone Reference Design.

The player control is a vertical, right aligned object occupying the unused space on album, artist and playlist views, while still allowing viewing and selection of the list items. It gives the user the ability to control playback without navigating away from the currently viewed list

The Album cover associated with the playing song is displayed if available.

Selecting Play from the Floating controls will display the Now Playing View and begin playing.

Selecting Add To Queue will add the Album to the current play-list.

View Album contents with Flow View

This feature is provided as part of the Deskphone Reference Design.

This view will display a rotary style view of album covers, allowing the user to select their next songs similar to a classic jukebox.

Selecting an album will display the album list view..

Selecting the Artist button will display the Artist List View

Selecting the Playlist View will display the Playlist VIew

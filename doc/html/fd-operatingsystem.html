Enablers: Operating System
Functional Description

Currently, Qt Extended only supports Linux.

Features

  • Display Driver
  • Keyboard Driver
  • Tool-chain
  • Touchscreen Driver

Display Driver

Qt Extended supports the standard Linux framebuffer. It is also possible to drive custom or accelerated hardware.

Keyboard Driver

Qt Extended supports standard keyboard drivers. It requires distinct press and release events (otherwise press-and-hold functionalities is lost). Auto-repeat can be handled by the kernel driver (in which case event timestamps must be coincident), or within Qt Extended.

Tool-chain

The GCC tool-chain is supported for cross-compiling. For Qt Extended 4, GCC version 4.1 is recommended. High quality tool-chains will include remote debugging gdb tools.

Touchscreen Driver

Qt Extended supports a number of touchscreens, including those supported by tslib.

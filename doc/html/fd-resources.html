LED Indicator Support

The Qt Extended API provides a set of classes that allows developers to modify the state of a device's LED indicators in a consistent manner.
The API is customizable and extensible to allow for specific LED capabilities such as:

  • Multiple Colors
  • Flashing
  • Throbbing
  • Brightness level

Qt Extended integration guide

Most parts of Qt Extended's documentation are designed to provide a reference for developer's. It can be difficult to read these references because they are taken out of context. The context is provided by the Qt Extended integration guide. It simplifies the integration process by providing a chronological order of tasks. The various tasks are put into an order and each task is briefly introduced (with more link to follow up). This improves the integration job by assisting during the planning phase and avoids problems later on due to forgotten tasks.

Note that the guide does not provide in-depth knowledge for each individual step. It rather aims to introduce the reader to a particular task and subsequently refers the reader to other parts of the Qt Extended documentation and/or external documentation.

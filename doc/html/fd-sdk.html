Enablers: SDK
Functional Description

The Qt Extended SDK provides developers with libraries and tools with which to develop software for devices running Qt Extended.

The Qt Extended libraries provide APIs for:

  • UI Controls & Interaction
  • Media and Document Handling
  • Input/Output
  • Data Structures
  • System access

Tools include:

  • Qt Designer
  • Qt Assistant
  • Qt Linguist
  • lrelease
  • lupdate
  • qmake
  • moc
  • uic

Source packages include the sources for the tools and script to build them. In the SDK packages all the above applications are provided as pre-built binaries.

Features

  • Neo SDK

Neo SDK

The Neo SDK is a complete SDK system, running in a vmware virtual machine. It allows developers to develop and debug Qt Extended applications for the FIC Neo on Windows, Linux and Mac OSX.

Developers can update Qt Extended on the device. It also has a mature KDevelop IDE.

The SDK parts can be easily installed when the developer already has an installed Linux desktop machine.

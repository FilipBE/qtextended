How to Obtain Qt Extended Products

Both licensed and open-source editions of Qt Extended are available. Details of how to obtain these editions are provided below. For specific licensing information please refer to: Qt Extended Licensing.

Open-source Editions

Qt Extended packages for the Open Source community can be downloaded from the Qt Extended Community Site.

Licensed Editions

After becoming a Qt Extended customer an email will be sent from qt-sales@nokia.com confirming your license details. The details include:

  • Customer ID
  • License ID
  • Qt Extended package download FTP site location.

Once you have received these details login to the FTP site using any browser and download the packages. Depending on your license the FTP site provides either a:

  • Qt Extended source package
  • Qt Extended SDK package

The FTP directory is available for the full duration of the contract, and will be updated regularly with new releases. Customers are notified via email about each new release that is relevant to their account.

Note: The FTP site may also provide access to other files such as various Qt versions, snapshots of packages, language packages and much more, however, this document refers only to the Qt Extended specific-files available on the FTP site.

Package Naming Conventions

The FTP download directory contains multiple packages and at first it may be unclear which packages are required. Depending on the type of license you have, one or more of the following packages is needed:

PackageDescription
qt-extended-[commercial|opensource]-src-[x.y.z].tar.gzFull source package: this package contains all sources to build Qt Extended, including the prerequisite source for Qt.
qt-extended-device-[devicename]-src-[x.y.z].tar.gzDevice source package: this package contains additional sources that are required to build Qt Extended for a specific device. A device package is installed 'on top' of the Qt Extended source package. Multiple device packages can be installed. Note that a number of device profiles are shipped in the standard source packages.
qt-extended-language-[languagename]-[x.y.z].tar.gzLanguage package: this package contains translations for a specific language. A language package is installed 'on top' of the Qt Extended source package. Multiple language packages can be installed.
qt-extended-[thirdparty-addon]-[commercial|opensource]-src-[x.y.z].tar.gzThirdparty add-on source package: this package contains sources for a thirdparty add-on module that can be used to enhance Qt Extended. A thirdparty add-on package is installed 'on top' of the Qt Extended source package. Multiple add-on packages can be installed.
qt-extended-[devicename]-sdk-[x.y.z].isoSoftware Development Kit (SDK) package: this package contains a combination of pre-built binaries and library header files. The iso package can be burned onto a CD and can then be installed on Windows and/or Linux operating systems. The SDK runs inside a VmPlayer environment.

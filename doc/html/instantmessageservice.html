Detailed Description

Provides the Qt Extended InstantMessage service.

The InstantMessage service enables applications to access features of the system's instant messaging application.

Member Function Documentation

void InstantMessageService::writeMessage ( const QString & uri )   [slot]

Direct the InstantMessage service to interact with the user to compose a new instant message, and then, if confirmed by the user, send the message. The message is sent to uri.

This slot corresponds to the QCop service message InstantMessage::writeMessage(QString).

Greenphone Performance Test Results

Introduction

This page contains performance statistics for Qt Extended, measured using the greenphone device profile. All figures are measured with respect to the default configuration.

Device specs

TypeValue
ModelGreenphone
KernelLinux 2.4.19
CPUIntel(R) XScale(R) PXA270 (ARMv5TE) 312 MHz
BasebandBroadcom(R) BCM2121 GSM/GPRS baseband processor
Memory64MB RAM & 128MB Flash
DisplayQVGA(R) LCD color screen
InputTouch-screen and keypad UI
FeaturesMini-SD(TM) card slot, Bluetooth(R) equipped, Mini-USB port

Testing takes place under the following conditions:

  • A SIM is inserted into the phone.
  • The phone is operating with a USB cable attached.

For Qt Extended 4.4.2 and later, Qt Extended and the root filesystem are prelinked. No prelinking was done in earlier versions.

Nokia N810 Performance Test Results

Introduction

This page contains performance statistics for Qt Extended, measured using the n810 device profile. All figures are measured with respect to the default configuration.

Device specs

TypeValue
ModelNokia N810
KernelLinux Nokia-N810-2.6.21-omap1

Display800x480 LCD color screen

Features802.11b/g, Bluetooth(R), Mini-SD(TM), GPS

Testing takes place under the following conditions:

  • The device is operating on a fully charged battery.
  • The device's keyboard is exposed (meaning Qt Extended is running in landscape mode).

Test Results

Linux + Qt Extended Startup Time

The Linux + Qt Extended startup time refers to the amount of time taken for a device to reach the Qt Extended homescreen and accept user input from the time the power button is pressed.

VersionTime (seconds)
4.4.1107
4.4.3115

The N810 root filesystem contains many services which are not used for Qt Extended. Therefore the total time to launch Qt Extended on the N810 is considerably longer than it would be if the root filesystem were optimized for Qt Extended.

Application Startup

The application startup time refers to the amount of time taken for an application to appear on the screen and respond to user input, after the user requests the application to launch.

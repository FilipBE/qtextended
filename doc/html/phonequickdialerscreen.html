Detailed Description

The PhoneQuickDialerScreen class implements a keypad based dialer UI.

An image of this dialer screen can be found in the server widget gallery.

This class is a Qt Extended server widget. It is part of the Qt Extended server and cannot be used by other Qt Extended applications.

See also QAbstractServerInterface and QAbstractDialerScreen.

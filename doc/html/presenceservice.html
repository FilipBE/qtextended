Detailed Description

The PresenceService class provides the Presence service.

The Presence service enables applications to modify the local presence.

Member Function Documentation

void PresenceService::dnd ()   [slot]

Toggles between Do not disturb state and online state. If the current presence is available, change it to away. It also updates the profile to the silence mode and direct calls to the voice mail.

This slot corresponds to the QCop service message Presence::dnd().

void PresenceService::editPresence ()   [slot]

Shows options to edit local presence status.

This slot is corresponds to the QCop service message Presence::editPresence().

Detailed Description

The QBluetoothHandsfreeAudioGatewayServer class implements the Bluetooth handsfree service Audio Gateway interface.

The QBluetoothHandsfreeAudioGatewayServer class implements Bluetooth Handsfree Audio Gateway profile..

QBluetoothHandsfreeAudioGatewayServer implements the QBluetoothAudioGateway interface. Client applications should use the QBluetoothAudioGateway class instead of this class to access the Bluetooth Audio Gateway functionality.

Please note, this class only acts as a forwarding agent to the QBluetoothHandsfreeService class, which handles all implementation details.

This class is part of the Qt Extended server and cannot be used by other QtopiaApplications.

See also QBluetoothAudioGateway and QBluetoothHandsfreeService.

i18n Extension

The i18n extension handles translatable files.

The usual way to enable i18n is to set the string language and the translation languages.

    STRING_LANGUAGE=en_US
    LANGUAGES=en_US de

AVAILABLE_LANGUAGES
EXTRA_TS_FILES
LANGUAGES
STRING_LANGUAGE
TS_DIR

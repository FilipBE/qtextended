fetchdata(obj, data): bool

Use it like this:

    var obj = project.property("target");
    var data = {
        files: {
            value: null,
            type: "existingFiles"
        },
        path: {
            value: null,
            type: "imagePath"
        }
    };
    if ( !installs.fetchdata(obj, data) ) return;

type can be one of:

  • single

    Returns a single value

  • array

    Returns an array of values

  • existingFile

    Returns a single, existing file

  • existingFiles

    Returns multiple, existing files

  • imagePath

    Returns a path in the image

isDirectory(file): bool

Pass in a filename (string) and returns true if it's a directory.

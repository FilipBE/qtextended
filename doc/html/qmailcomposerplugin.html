Detailed Description

The QMailComposerPlugin class defines a base class for implementing mail message composer plug-ins.

The QMailComposerPlugin class provides a base class for plug-in classes that provide mail message composing functionality. Classes that inherit QMailComposerPlugin need to provide overrides of the keys() and create() member functions.

See also QMailComposerPluginInterface and QMailComposerInterface.

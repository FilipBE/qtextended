Detailed Description

The QModemCallSettings class implements access to auxiliary call settings for AT-based modems.

This class uses the AT+CCWA, AT+CLIR, AT+CGSMS, AT+CLIP, AT+COLP commands from 3GPP TS 27.007 and 3GPP TS 27.005.

QModemCallSettings implements the QCallSettings telephony interface. Client applications should use QCallSettings instead of this class to access the modem's call settings.

See also QCallSettings.

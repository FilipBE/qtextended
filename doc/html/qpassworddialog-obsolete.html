void QPasswordDialog::authenticateUser ( QWidget * parent, bool atPowerOn = false )   [static]

Prompt, fullscreen, for the user's passcode until they get it right.

If atPowerOn is true, the dialog is only used if the user's preference request it at poweron.

bool QPasswordDialog::authenticateUser ( const QString & text )   [static]

This is an overloaded member function, provided for convenience.

Returns true if text matches the user's passcode; otherwise returns false.

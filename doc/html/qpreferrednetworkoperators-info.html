Detailed Description

The Info class specifies information about a preferred network operator on the SIM.

The index field indicates the position within the list (1 indicates the most-preferred operator). The format field indicates the 3GPP TS 27.007 format of the operator name (0 = long alphabetic, 1 = short alphabetic, 2 = numeric). The id field indicates the numeric operator code, if known. The name field indicates the alphabetic operator name, if known. The technologies field indicates the list of operator technologies that are preferred (GSM, GSMCompact, UTRAN, etc).

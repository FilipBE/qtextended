500

Usage

This command includes an untranslated error message that can be ignored.

    500 Syntax error, command unrecognized

Description

Sent from either the device or desktop. This is sent in response to a CALL command with an invalid type or to a CALL or CALLB command with less than 2 arguments.

See also qdsync Protocol.

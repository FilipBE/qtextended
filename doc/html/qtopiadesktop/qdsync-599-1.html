599

Usage

This command includes an untranslated error message that can be ignored.

    599 ChannelNotRegistered QPE/Channel
    599 send failed

Description

Sent from either the device or desktop. This is sent in response to a CALL or CALLB command when the destination channel does not exist or the QCopChannel::send() function returns false. As a special case, messages sent to QPE/Application/* are always delivered because the application may not be running when the command is sent.

See also qdsync Protocol.

Detailed Description

The QWmlReader class reads Wireless Markup Language (WML) documents

The QWmlReader class extends QWbXmlReader to provide support for reading documents in the Wireless Markup Language (WML). This language is defined in the WAP standard wap-238-wml-20010911-a.pdf.

See also QWbXmlReader.

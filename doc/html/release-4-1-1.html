Qtopia - 4.1.1 Release Notes

  • Introduction
  • Enhancements

Introduction

The Qtopia 4.1.1 release is a patch release that provides:

  • fixes for a number of device build issues
  • improved Security profile
  • updated documentation.

Enhancements

  1. Devices
    • added a example Device Profile
    • moved phone vendor/multiplexer plug-ins into the devices section and changed when they are built
    • fixed singlexec building for devices.
  2. Documentation
    • updated the SXE profiles
    • improved documentation for devices
    • fixed/clarified 3rdparty GPL licensing issues.
  3. Cleanup
    • removed obsolete rsync code.

Detailed Description

The ThemeSetupTask class ensures a theme preselection for Qtopia.

The selection process happens in the following order:

  1. Use the theme specified in $HOME/Settings/Trolltech/qpe.conf using the Appearance/Theme key (user selected theme).
  2. Use the theme specified in Qtopia::qtopiaDir()/etc/default/Trolltech/qpe.conf using the Appearance/Theme key (Qt Extended default theme).
  3. Pick an arbitrary theme from Qtopia::qtopiaDir()/etc/themes/
  4. Don't use any theme.

The ThemeSetupTask class provides the ThemeSetup task. It is part of the Qt Extended server and cannot be used by other Qt Extended applications.

Show Groups

Use this feature to manage groups of contacts. While showing the list of groups it is possible to:

  • Use the New group option to create a new group. The new group will initially be empty. Select the group for more options.
  • Use the Send all option to send all the contacts in the group to another device. (See Sending Contacts)
  • Use the Set group ringtone option to assign a ringtone for the entire group. Contacts with individual ringtones set will still have the contact ringtone used in preference to the group ringtone.
  • Use the back button to return to the list of all contacts.

After selecting a group it is possible to:

  • Use the / buttons to show different groups.
  • Use the back button to return to the list of groups.
  • Use the Manage group members option to add or remove members from the current group.

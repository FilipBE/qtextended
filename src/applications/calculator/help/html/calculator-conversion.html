Conversion Mode

This mode allows you to convert temperatures, area, distance, volume and weight measurements from one standard to another.

To convert a measurement:

  • Select the measurement type from the drop-down list (eg temperature).
  • Select the standard measurement of the original to be converted (eg celsius).
  • Enter the numbers of the measurement (eg 36).
  • Tap = to indicate that this number is to be converted.
  • Tap the button with the new standard (eg fahrenheit).
  • The converted number is displayed.
  • Tap CE/C to clear the entry field.

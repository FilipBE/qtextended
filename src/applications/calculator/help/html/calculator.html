Calculator

A basic four-function calculator is available with this device.

To enter a number: use number keys.

To apply a function:

  1. + is
  2. - is
  3. / (divide) is
  4. x (multiply) is
  5. * is decimal point
  6. OK is =

To change between operators: use the # button.

To clear the calculation after pressing OK, press 0.

  • M+ adds the value in the numeric display to the current value in memory. (An "m" appears in the numeric display when the memory contains a value. This will disappear when the memory is cleared.)
  • MC clears the calculator memory.
  • MR recalls the current value from memory.
  • CE/C Tapped once clears the current entry in the numeric display. Tapped twice clears all values entered for this calculation.
  • <- deletes the last entered digit.

The Calculator functions in the same way as a handheld calculator. Four calculator modes are available from the drop-down list below the numeric display.

  • Fraction Mode
  • Conversion Mode
  • Simple Mode
  • Scientific Mode

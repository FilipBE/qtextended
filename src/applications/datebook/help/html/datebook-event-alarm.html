Event Alarm

Event Alarms are used to remind you of upcoming events. They can be Silent (a message box will appear when the alarm is activated) or Audible (a tone will sound).

To set a reminder for an event:

  • Select either Silent Alarm or Audible Alarm from the alarm drop-down list.
  • Select how long before the event the reminder should occur.
  • If you have an all day event, you can also select what time of day you would like the reminder to occur.

Composing MMS Messages

MMS messages can carry multiple media items. An MMS message presents the media items in a slide-show format, where each slide can contain a single image, text field and audio clip.

Image: use the Slide photo dialog to choose an image to add to the slide. Press Pictures to choose an image from existing documents, and Edit image to make changes to the image. When you are happy with your selection, press Back to add the image to the slide.

Text: enter the text you wish to send.

Audio: use the Slide audio dialog to choose an audio clip to add to the slide. Press Audio Files to choose from existing documents, or Record Audio to record a voice message. When you are happy with your selection, press Back to add the audio clip to the slide.

Multiple Slides

To add a new slide to your slide-show, press Options and select Add Slide. The text in the top-right corner will note which slide you are currently editing.

Use / to navigate between the slides in your slide-show.

To remove the current slide from your slide-show, press Options and select Remove Slide.

Slide Options

To change the properties of your slide-show, press Options and select Slide Options....

Duration: set how long each slide in your slide-show will display for, before the next slide is shown.

Slide color: set the background color that will be shown behind your slide elements.

System Info

This application presents a set of tabs each containing information about the current status of your .

Version: version information for the software build installed on your .

Storage: used and available space on the device's internal storage and any removable storage cards. The storage tab also provides a cleanup wizard in order to free up space on the device.

CPU: shows a graph of recent CPU utilization.

Memory: free and used amounts of memory.

Data: provides a detailed breakdown of the storage being used for your documents.

Security: the state of the safe execution environment.

Modem: technical data about the device's GSM modem.

SIM: the number of used and free SMS and Contacts slots on the SIM card.

Deleting Categories

Categories can only be deleted from an application that uses categories. Default categories cannot be deleted.

Items in a deleted category become Unfiled.

Delete a Category

  • Open the Documents list.
  • Highlight any document.
  • Press Options and select Properties.
  • Press the Category button.
  • When the list of available categories is displayed, highlight the category to be deleted.
  • Press Options and select Delete.
  • Confirm the deletion.
  • Press Back to save and exit.
  • Press Back again to exit Documents.

Home Screen

The home screen is the gateway to all of Qt Extended's features.

From here you can:

  • Make Calls.
  • Enter a number then store it (saving to Contacts).
  • Enter a number and send an SMS (using the Messages application).
  • Look-up a contact by entering part of their number or name.
  • Launch the most commonly used applications (select them by using the up and down arrow keys).
  • View today's appointments.
  • View the current time in any time zone by using the world map.
  • Lock the device against unwanted input by selecting Key Lock from the Options menu.
  • Lock the device against unwanted input by tapping the padlock icon.
  • Quickly change the current profile by selecting Profiles from the Options menu.
  • Quickly view missed calls or new messages by selecting Missed Calls or New Messages from the Options menu.
  • Access online help by selecting Help from the Options menu.
  • Access other functionality by selecting Menu.

The home screen also shows the current profile and other phone status information such as if an alarm has been set.

Appearance

Use Appearance to change your device's visual appearance including:

  • Theme: change the look of homescreen, title bar and soft menu bar.
  • Color: choose a color scheme.
  • Background: Choose a background image supported by the selected theme.
  • Menu bar: choose text or icons for the soft menu bar at the bottom of the screen.

The changes will be applied when you exit the application.

Additionally, the following menu options are available:

  • Add to current profile - Add the current appearance state to the active profile so that this state can be applied when the profile is activated.
  • Homescreen Settings - Open application to change the appearance of the homescreen.

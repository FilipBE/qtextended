Adding Characters

You can add new characters using the options in the Handwriting Settings application.

  • Select the type of screen and character from the list, eg. Full Screen Handwriting -> Numver.
  • Tap the new characters icon to open the dialog.
  • Select the character / function from the list.
  • Press Back to proceed.
  • Using the new character input area draw the character. The character indicator will change from 0/0 to 1/1 (1st of a total of 1).

You can now choose to keep the character as you have drawn it, try drawing it again and compare the two versions, or discard the attempt and start again.

  • To accept the new character, press Select and exit.
  • To draw another version of the character, tap and proceed to draw it again. The indicator will update to 2/2 (2nd of a total of 2).
  • To compare the two (or more) versions tap the / arrows located next to the input area.
  • To discard a version, display it then tap to remove it.
  • Press Select to accept the new character and Back to exit.

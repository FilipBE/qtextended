Bluetooth DUN Parameter

Complete the form by entering the information into the following fields:

The dial string is required for every connection. A typical dial string would be

*99***1#

The second field can contain additional AT commands which are issued before the dial command is executed. Usually this is used to select the APN and the transfer protocol. Each command must be on a new line.

A typical APN selection may look like this:

+CGDCONT=1,"IP","internet"

Note: The dial command must not include the AT prefix.

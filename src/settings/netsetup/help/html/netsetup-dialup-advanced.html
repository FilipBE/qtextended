Advanced Options

The following options allow a more detailed configuration of the network service. Usually it should not be required to change these parameters.

Auto routing: the network service is added as the default route to the system routing tables

External serial device: if this option is ticked you can manually enter a device node to the serial device, e.g. /dev/ttyS1. If this option is un-ticked the service will use the internal modem for a data connection.

AT-dial: the AT-dial dial code supplied by your ISP, e.g. ATDT.

Speed: at which modem should connect (influenced by ISP).

Flow Control data transfer protocol. Recommended.

Wait Time acceptable delay for other end of connection to communicate. If data transfer to device is delayed more than this time, then auto disconnect. Use the default 5 seconds, or increase the time if you have problems connecting.

Silent dial tone: suppresses the dial tone if selected.

OpenVPN Account Details

Used to define general OpenVPN account details.

Complete the form by entering the information into the following fields:

Name: the name you wish to give this account.

Description: a general description of this account.

Remote: the remote OpenVPN server you wish to connect to.

Ping: pings remote if no packets have been sent for the given amount of seconds.

Ping Restart: restarts OpenVPN if no pings/packets have been received for this amount of time.

Wireless Scan

Qt Extended can actively scan for surrounding wireless networks. Every network that is selected will automatically added to the list of configured networks.

The priority determines the priority of configured/known WLANs. When attempting to connect (by means of manual or automatic (re)connect) to a WLAN the device connects to the WLAN with the highest priority first. The list uses a descending order (first WLAN has highest priority).

The order can be changed by selecting the WLAN using Select, then use   to move the selected entry up/down and finally lock the selected entry in the desired spot in the list be pressing Select again.

The form provides the following context menu options:

  • Rescan: initiates a new scan
  • Add new networks: shows the result of the last network scan
  • Connect: connect to the selected network
  • Delete: deletes the selected network.

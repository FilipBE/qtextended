Install Application

  • Select the Downloads tab.
  • Select Connect from the Options menu to be shown a list of available servers.
  • Select a server to connect to it. A list of available applications is displayed and the connection's status is shown in the bottom pane.
  • Highlight an application and select Install from the Options menu. You will be asked to confirm installation.
  • To confirm installation select Confirm from the Options menu. A successful installation dialog will be presented. Press Ok and the newly installed package will be highlighted in the Installed tab. (The application will no longer be available in the Downloads tab.)
